# Huayuchen Hua — Academic Homepage

Personal academic homepage of [Huayuchen Hua](https://CHERRY-ui8.github.io/academic).

Built with pure HTML + CSS. No build tools, no frameworks. Hosted on GitHub Pages.

## Pages

- **Home** (`/`) — About, Research Interests, News
- **Research** (`/research`) — Interests, Philosophy, Ongoing Work
- **Publications** (`/publications`) — Published papers
- **Projects** (`/projects`) — Research project details
- **CV** (`/cv`) — Experience, Education, Awards, Skills, Leadership
- **Contact** (`/contact`) — Contact information and links

## Local Development

No build step needed. Just open any `.html` file in your browser.

```bash
python3 -m http.server 8000
# or
python -m http.server 8000
```

## Deploy

Push to `main` branch. GitHub Pages serves from the root automatically.
